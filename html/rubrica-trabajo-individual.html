Uso el PCBusca la información en la web elegida y navega sin ayuda. (2.5)Encuentra la información con un poco de ayuda del profe. (1.75)Necesita que el profe le diga todo el tiempo dónde clicar. (1.50)No sabe usar la página web para buscar la leyenda. (1.25)
